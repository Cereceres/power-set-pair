# power-set-pair
iterate over a array to get all subsets

# Installation

```bash
npm install --save power-set-pair
```
# Usage

```js
convolution([1, 2, 3, 4], (a, b) => a + b) // [3, 4, 5, 5, 6, 7]
```

# API

## powerSetPair(array, mapper)-> array


## mapper(arrayElement, otherArrayElement) -> Any
